# T009: Kilo CLI Error

## Context
Verify handling of Kilo CLI failures.
This task should cause Kilo to error.

## Test Specification

### Expected Flow
1. State: IMPLEMENTING
2. Kilo command fails or returns error

5. Eventually circuit breaker

Exit with error to simulate Kilo CLI failure:
```bash
echo "Simulating Kilo CLI error" >&2
exit 2
```
