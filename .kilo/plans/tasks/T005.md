# T005: Multiple Parallel Tasks

## Context

Verify Ralph Loop handles sequential task execution after previous tasks completed.
Create a simple file creation task that review should APPROVE.

## Test Specification

### Expected Flow
1. Ralph Loop detects T005 as next executable (no dependencies)
2. State: IMPLEMENTING
3. Kilo creates file: test_output_t005.txt with "Hello from T005"
4. Creates `.ralph_pending_tasks.json` with task_id: "T005"
5. State: REVIEWING
6. Reviewers APPROVE
7. Task marked [x] in tasks.md
8. State: IDLE

### Validation
- `.ralph_pending_tasks.json` created with task_id: "T005"
- tasks.md shows `- [x] T005`
- test_output_t005.txt exists with correct content

## Implementation Instructions

Create a simple file:
```bash
echo "Hello from T005" > test_output_t005.txt
```

Then create pending file:
```bash
cat > .ralph_pending_tasks.json << 'JSON'
{"task_id": "T005", "files_changed": ["test_output_t005.txt"], "summary": "Created test file for T005"}
JSON
```

## Acceptance Criteria
- [ ] Task executed successfully
- [ ] test_output_t005.txt created
- [ ] Review APPROVED
- [ ] State transitions to IDLE
