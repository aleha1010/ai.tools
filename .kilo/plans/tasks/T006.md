# T006: Circuit Breaker (3 Consecutive Failures)

## Context
Verify circuit breaker stops after 3 consecutive failures.
This task is designed to fail repeatedly.

## Test Specification

### Expected Flow
1. Iteration 1: T006 fails → consecutive_failures=1, backoff 2s
2. Iteration 2: T006 fails → consecutive_failures=2, backoff 4s
3. Iteration 3: T006 fails → consecutive_failures=3, backoff 8s
4. Circuit breaker triggers
5. State: FAILED
6. Exit with error

## Implementation Instructions

This task should exit with non-zero code:
```bash
echo "Simulating failure" >&2
exit 1
```

## Acceptance Criteria
- [ ] consecutive_failures increments correctly
- [ ] Exponential backoff: 2s, 4s, 8s
- [ ] Circuit breaker at 3 failures
- [ ] State: FAILED
- [ ] Error message: "Circuit breaker"
