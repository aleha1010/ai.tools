# T002: Review Rejection + Retry

## Context

Task should be REJECTED first time, then APPROVED after fix.

## Test Specification

1. State: IMPLEMENTING (T002)
2. Create code with intentional security issue (hardcoded password)
3. State: REVIEWING
4. review-security detects vulnerability → REJECTED
5. State: REJECTED
6. `.ralph_rejection_context.md` created

1. Read `.ralph_rejection_context.md`

5. All reviewers APPROVE
